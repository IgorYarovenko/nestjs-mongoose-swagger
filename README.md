## Installation

```bash
$ npm install
```

## Running the app
```bash
create .env file from .development.env

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Database

```bash
cd ./mongodb
docker-compose up -d
```
